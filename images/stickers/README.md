# Replace these with your actual banana sticker images

This folder should contain your banana sticker images in JPG, PNG, or WebP format.

## Image Guidelines:
- Recommended size: 400x400px (square aspect ratio works best)
- File format: JPG, PNG, or WebP
- Keep file size under 200KB for optimal performance
- Use descriptive names like: banana-happy.jpg, banana-cool.jpg

## How to add images:
1. Save your images to this stickers folder
2. Update the src paths in index.html to match your image filenames
3. Update the alt text and titles in index.html
